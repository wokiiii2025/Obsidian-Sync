# Obsidian Server

This repository contains two deliverables described in `Docs/obsidian-sync-design.md`.

- `server/` - FastAPI + PostgreSQL sync server
- `client/obsidian-plugin/` - Obsidian desktop/mobile plugin

## Current Status

- Phase 1 server scaffold is implemented.
- Phase 2 client plugin scaffold is implemented with encryption and sync API plumbing.
